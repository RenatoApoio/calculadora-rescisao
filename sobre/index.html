💡 O que é o FacilCalc?

O FacilCalc nasceu de uma necessidade simples: quando as pessoas precisam calcular rescisão, FGTS, IR ou parcelas de empréstimo, muitas vezes não têm todas as informações em mãos.

A maioria das calculadoras na internet exige todos os dados preenchidos ou apresenta resultados difíceis de entender. O FacilCalc foi criado com um diferencial: "calcule com o que tem" — estimativas automáticas preenchem o que falta, e tudo é explicado em linguagem simples.

🛠️ Como foi feito

O FacilCalc é um projeto desenvolvido por um programador brasileiro independente, utilizando apenas HTML, CSS e JavaScript puro — sem frameworks ou bibliotecas externas.

O site é hospedado no GitHub Pages (gratuito) com domínio próprio facilcalc.com.br. Todo o código é executado diretamente no navegador do usuário — nenhum servidor próprio processa seus dados.

As calculadoras são baseadas na legislação vigente (CLT, tabelas INSS/IR 2025, EC 103/2019) e são atualizadas regularmente.
